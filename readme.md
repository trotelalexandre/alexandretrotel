# My Portfolio

Welcome to my portfolio! Here you can find a showcase of my projects and my journey on how I made this portfolio.

## About

This portfolio is created using Svelte, a fantastic JavaScript framework that allowed me to build efficient and interactive user interfaces. The site is built with a mobile-first approach, which means it is responsive and optimized for different screen sizes. The site is hosted on Vercel, providing a seamless and reliable deployment experience.

## How I Made This Portfolio

I created this portfolio using the following steps:

1. Planning: I started by defining the goals of my portfolio, identifying my target audience, and creating a wireframe of the site.

2. Design: I created a design that is visually appealing and consistent with my personal brand.

3. Development: Leveraging the power of Svelte, I efficiently implemented the user interface and functionality, utilizing HTML, CSS, and TypeScript.

4. Deployment: Thanks to Vercel's user-friendly platform, I seamlessly deployed the site, making it accessible to visitors from all around the world.
